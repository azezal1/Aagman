# Assets Folder

Place your app assets here:

## Images
- `images/logo.png` - App logo for splash screen
- `images/bus_icon.png` - Bus illustrations (optional)

## Icons
- Custom icons if needed (the app uses Material Icons by default)

## Recommended Sizes

- Logo: 512x512px PNG with transparency
- Icons: 24x24dp, 48x48dp, 72x72dp

## Note

The app currently uses Material Icons for all UI elements to maintain a clean, minimal aesthetic. Custom assets are optional.
